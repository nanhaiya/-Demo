### 1，首先在讯飞开放平台注册和申请用户并实名认证  
### 2，申请应用，获得密匙  
### 3，不同功能需求有不同密匙，当前密匙为语音转写密匙  
### 4，语音转写密匙需要在配置文件（config.properties）配置密匙：app_id、secret_key  
### 4，变量local_file为原始音频存放地址，需要自行更改  
### 5，转文本生成的txt文件和原始音频地址一致，暂时未做自定义选择目录功能，需要自行修改
